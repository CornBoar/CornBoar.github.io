CornBoar API

API for the YBA Bot and the Demon List Verifications Discord Server. Base URL is https://api.cornboar.com/.

YBA Bot Endpoints:

● /stand/{user_id}/ - Get a user's equipped stand.

● /stands/{user_id}/ - Get all of the stands of a user.

● /items/{user_id}/ - Get all of the items of a user.

● /pity/{user_id}/ - Get the pity of a user.

● /username/{user_id}/ - Get a user's username by their ID.

● /is_admin/{user_id}/ - Check a user's admin status.

Demon List Verifications Endpoints:

● /dlvlist/ - Get an ordered list of the demons, their victors, and their colors.

● /dlvusers/ - Get the stats of all users.

● /dlvusers/{user_id}/ - Get the stats of a user.

● /dlvuserstop100/ - Get the stats of the top 100 users ordered by XP.

● /dlvvalidatekey/{key}/ - Check if an admin website key is valid.

● /dlvadddemon/{key}/{position}/{name}/{color}/ - Add a demon to the list.

● /dlvremovedemon/{key}/{name}/ - Remove a demon from the list.

● /dlvaddcompletion/{key}/{user_id}/{demon_name}/ - Add a completion to a user.

● /dlvremovecompletion/{key}/{user_id}/{demon_name}/ - Remove a completion from a user.

● /dlvsetxp/{key}/{user_id}/{xp_amount}/ - Edit a user's XP.

● /dlvgetpoints/{level}/ - Get a level's XP value.

Keys are obtained through the DLV Bot, can only be obtained by CornBoar and Dr. Slug.
